HyperText Markup Language

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat halaman web. HTML tidak memiliki kapabilitas pemrograman seperti bahasa pemrograman lainnya, melainkan hanya memberikan tata letak dan struktur bagi konten yang ingin ditampilkan di halaman web.

HTML menggunakan tag (misal: <"p">, <"h1">, <"a">) untuk menentukan bagaimana konten harus ditampilkan. Setiap tag memiliki arti yang berbeda dan mempengaruhi bagaimana konten tersebut ditampilkan.

Contohnya, tag <"p"> digunakan untuk memformat teks sebagai paragraf, tag <"h1"> digunakan untuk memformat teks sebagai judul utama, dan tag <"a"> digunakan untuk membuat tautan ke halaman web lain.

HTML juga memiliki atribut (misal: class, id, style) yang dapat digunakan untuk memberikan informasi tambahan tentang bagaimana tag tersebut harus ditampilkan.

Dengan HTML, Anda dapat membuat halaman web yang dinamis dan interaktif dengan menambahkan elemen seperti gambar, tabel, formulir, dan media. HTML adalah bahan dasar bagi pengembangan halaman web dan sering digunakan bersama dengan bahasa pemrograman seperti JavaScript dan CSS.

Cascading Style Sheet

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk memformat dan memperindah tampilan halaman web. CSS memungkinkan Anda untuk memisahkan presentasi dari isi halaman web, sehingga lebih mudah untuk memelihara dan memodifikasi tampilan situs.

CSS menggunakan aturan (rules) yang diterapkan pada elemen HTML untuk menentukan bagaimana elemen tersebut harus ditampilkan. Aturan ini berisi properti (misal: font-size, color, background-color) dan nilai (misal: 16px, blue, red) yang menentukan bagaimana elemen tersebut harus ditampilkan.

CSS juga memiliki konsep seperti inheritance dan cascading, dimana aturan diterapkan dari parent ke child element dan aturan yang lebih spesifik akan menimpa aturan yang lebih umum.

Dengan CSS, Anda dapat mengontrol tampilan seluruh situs secara massal, membuat situs yang responsif untuk berbagai ukuran layar, dan menambahkan interaksi dan animasi melalui transisi dan transformasi. CSS adalah komponen penting dalam pengembangan halaman web dan sering digunakan bersama dengan HTML dan JavaScript.

JavaScript

JS (JavaScript) adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif dan dinamis. JavaScript memungkinkan Anda untuk menambahkan logika dan interaksi pada halaman web, seperti membuat tombol yang dapat diklik, memvalidasi formulir, dan membuat animasi.

JavaScript berjalan di luar browser (client-side) dan memanipulasi halaman web secara real-time. Ini berarti bahwa JavaScript dapat membuat halaman web merespons aksi pengguna seperti klik tombol, geser, dan mengetik, tanpa harus memuat halaman ulang.

JavaScript memiliki sintaks yang mirip dengan bahasa pemrograman lain seperti C dan Java, dan menyediakan fitur seperti variabel, tipe data, operator, perulangan, dan fungsi.

JavaScript juga memiliki ekosistem yang kuat dengan berbagai library dan framework seperti jQuery, React, Angular, dan Vue.js, yang membuat pengembangan halaman web lebih efisien dan efektif. JavaScript adalah bahasa pemrograman utama bagi pengembangan halaman web dan sering digunakan bersama dengan HTML dan CSS.
